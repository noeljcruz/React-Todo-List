# Errand List

## SUMMARY

Errand List is a React Todo List application where you can keep all your errands and todo items organized in one convenient place!

Features include:

* Local Storage
* Add, Delete, Update
* Nested Items
* Collapsible Display
* Linked Parent/Child Completion
